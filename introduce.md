  架构概要

  用户 → Vue 3 前端 → SSE/HTTP → FastAPI 后端
                                      │
                          Orchestrator (RAG 编排)
                          ├── Main Agent (LLM + 3 Tools)
                          ├── Analysis Agent (意图分析)
                          ├── Command Generator (命令生成)
                          ├── RAG Retriever (ChromaDB 检索)
                          └── 后处理 (校验/布局/格式化)
                                      │
                                Ollama (Qwen3-4B + bge-m3)

  总计约 4,100 行代码，204 个文件（含 170+ 知识库 JSON）。
